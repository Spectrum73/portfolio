Unity Action Platformer

Skills used in this project

  • Self Learning
  • Unity and C#
  • 3D Modelling and Asset Creation

Summary

This is a personal project I've used to learn about Unity and game creation, I have also used it to learn lots about Blender for 3D asset creation and how to implement them in-engine. The game itself is an action platformer, featuring various movement options and combo oriented combat.

Player Controller

The player controller features a number of different movement options including: wallrunning, dashing and swinging from a grappling hook. The controller uses the Kinematic Character Controller Asset for its physics and collision handling.

Combat

Combat uses traditional fighting game combo principles, featuring cancels, specials and other systems. An action buffer is used to improve responsiveness. All systems were implemented from scratch so that I could learn how they should work in more detail whilst making them.

Attacks for the combat system use a modular approach, being constructed from multiple smaller actions playing at set times throughout the attack, some of these actions include: Activating a hitbox, playing VFX or changing the actor's velocity. Similarly, hitboxes are also modular, instances can easily have their hit data adjusted, including the number and placement of hitboxes for a hitbox type.
