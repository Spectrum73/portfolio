Summary

This is a personal project I've used to learn about Unity and game creation, I have also used it to learn about Blender for 3D asset creation, including how to export and implement them in-engine. My core goal with the project, other than learning more about Unity and Blender, was to create a game I felt proud of and would enjoy playing. The game itself is an action platformer, featuring fast, fluid movement and combo-oriented combat.

A common issue I run into with the project is scope, it's very ambitious and requires a lot of features to be developed for a single person. To combat this, I'll typically work on one feature at a time (movement, combat, AI, etc) and list features that need implementation, usually through GitHub Issues.

Demo Videos

Player Controller

The player controller features a number of movement options including: wallrunning, dashing and swinging from a grappling hook. It uses the Kinematic Character Controller Asset for its physics and collision handling.

Combat

Combat uses traditional fighting game combo principles, featuring cancels, specials and other systems such as an action buffer is used to improve responsiveness. Most systems were implemented from scratch so that I could learn how they should work in more detail whilst making them.
